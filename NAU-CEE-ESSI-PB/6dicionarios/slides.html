Definição

Um dicionário é um objeto composto por pares chave-valor. Dada uma chave, o dicionário permite encontrar rapidamente o valor associado. Como os elementos são identificados através da chave, a ordem é irrelevante. Mas cada elemento tem de ter uma chave única, distinta das restantes. Por isso dizemos que um dicionário é uma coleção associativa, não ordenada de elementos únicos.

Dicionário (= vetor associativo):
Coleção associativa, não ordenada de elementos únicos. 
capitais = {'Portugal': 'Lisboa', 'Angola': 'Luanda'}
  • Cada elemento é um par chave-valor.
  • A chave permite encontrar o valor associado.
  • A ordem dos elementos é irrelevante.
  • Todas as chaves são distintas.

Indexação segura

Indexar um dicionário com uma chave inexistente é um erro de tipo KeyError. Por isso, antes de indexar com uma chave desconhecida, devemos verificar se ela está no dicionário ou não. Os operadores in e not in servem para isso. Avance passo-a-passo para perceber como se usam.

Acesso seguro com get

Em alternativa à indexação podemos usar o método get, que devolve um valor mesmo que a chave não exista. Avance passo-a-passo para perceber como se usa.

Percorrer dicionários

Por vezes temos de percorrer sistematicamente todos os elementos de um dicionário. Podemos fazê-lo com um ciclo for e usar um método para gerar uma sequência das chaves, dos valores ou dos pares chave-valor.

Podemos usar for e um dos métodos abaixo.

Método Resultado
dict.keys() sequência das chaves
dict.values() sequência dos valores
dict.items() sequência de pares (chave, valor)
